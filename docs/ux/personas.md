# Personas

## 1. Brand Owner (Primary)

- Description:
  - Owns or manages a clothing brand.
  - Not very technical; comfortable with basic apps and WhatsApp.
- Goals:
  - Increase repeat orders and engagement after calls.
  - See call and WhatsApp follow-up performance.
  - Set up system once, then let it run automatically.
- Pain Points:
  - Forgets to follow up after calls.
  - No centralized view of calls + messages + products.
- Behaviours:
  - Uses WhatsApp a lot.
  - Works from laptop/desktop in the store or small office.
- Needs from the system:
  - Simple setup wizard.
  - Clear “ON/OFF” for automation.
  - Easy product data upload.


## 2. System Admin

- Description:
  - You, or your team, who create tenants and support clients.
- Goals:
  - Quickly onboard new clothing brands.
  - Debug issues (webhook issues, API issues).
- Needs:
  - A simple internal way to create tenants and owner accounts.
  - Access to logs and basic health checks (later phases).
